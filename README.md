## ⚡ Hakkında

**Nihat-Portifolio**, sıfırdan tasarlayıp geliştirdiğim kişisel portfolyo sitemdir. Modern web teknolojileri, gerçek zamanlı sohbet sistemi ve detaylı yönetim paneli ile donatılmıştır.

> Yolun başında olan bir öğrenci olarak hayal ettiğim, her detayını kendi ellerimle ördüğüm dijital evim.

## 📁 Proje Yapısı

```
Nihat-Portifolio/
│
├── index.html          # Tüm site - tek dosyada
├── README.md           # Dokümantasyon
└── ...
```

> 💡 **Tek dosya yaklaşımı:** Hızlı yükleme ve kolay deploy için tüm kod tek HTML dosyasında. Tailwind CSS ve Firebase CDN üzerinden çağrılıyor.

<br>

## 🚀 Yerel Çalıştırma

```bash
# Depoyu klonla
git clone https://github.com/mhmmtmst/Nihat-Portifolio.git

# Klasöre gir
cd Nihat-Portifolio

# index.html'i tarayıcıda aç
# (veya Live Server eklentisiyle çalıştır)
```
